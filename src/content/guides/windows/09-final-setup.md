После настройки VPN рекомендуется выполнить несколько финальных шагов.

### Автозапуск (v2rayN)

1. Перейдите в `Settings` → `Option Setting`
2. Во вкладке v2rayN включите `Start on boot` — приложение будет запускаться при старте системы
3. Включите `Auto hide startup` — приложение не будет отображаться поверх других окон при запуске и сразу свернётся в трей

### Обновления

1. На главном экране нажмите `Check Update`
2. Это скачает все обновления, включая GeoFiles, которые в дальнейшем будут обновляться автоматически каждые 12 часов
3. Обновления самого приложения проверяйте вручную раз в месяц

### Режим TUN (для продвинутых)

Если нужно пропустить через VPN **весь** системный трафик (включая игры и программы, которые игнорируют системный прокси):

1. Выберите `Clear system proxy`
2. Выберите правило `ProxyAll`
3. Включите переключатель `Enable TUN` — программа запросит права администратора и перезапустится

<div class="warning">

**Важно:** постоянно использовать режим TUN не рекомендуется, если он вам не нужен — он может создавать лишнюю нагрузку.

</div>

<div class="tip">

**Совет:** чтобы не перезапускать программу каждый раз при включении TUN, можно запускать v2rayN с правами администратора. Найдите файл `v2rayN.exe`, нажмите правой кнопкой мыши → `Совместимость` → `Запускать эту программу от имени администратора`.

</div>
