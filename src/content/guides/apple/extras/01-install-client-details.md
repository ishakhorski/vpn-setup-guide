В российском регионе App Store недавно были удалены все популярные приложения для VPN. Если ни один из предложенных клиентов не будет доступен — **скачайте любой клиент по запросу "Vless xray"**

### Есть несколько альтернативных клиентов

- Первый альтернативный клиент — **V2Box** отличный клиент с технической точки зрения, но присутствует реклама и есть сомнения в своевременной поддержке — <a href="https://apps.apple.com/us/app/v2box-v2ray-client/id6446814690" target="_blank">Перейти в App Store</a><QRCodeButton link="https://apps.apple.com/us/app/v2box-v2ray-client/id6446814690" />
