### Есть несколько альтернативных клиентов

1. Первый альтернативный клиент — **NekoBox**. Он доступен для скачивания на GitHub.
   Болле сложная но в то же время гибкое приложение на ядре sing-box.
   Главное отличие - меньшая нагрузка на процессор, но худшая совместимость с нашими серверами. Не рекомендуется для новичков.

**NekoBox arm64-v8a** — <a href="https://github.com/MatsuriDayo/NekoBoxForAndroid/releases/download/1.4.2/NekoBox-1.4.2-arm64-v8a.apk" target="_blank">Скачать с GitHub</a> <QRCodeButton link="https://github.com/MatsuriDayo/NekoBoxForAndroid/releases/download/1.4.2/NekoBox-1.4.2-arm64-v8a.apk" />

**NekoBox armeabi-v7a** — <a href="https://github.com/MatsuriDayo/NekoBoxForAndroid/releases/download/1.4.2/NekoBox-1.4.2-armeabi-v7a.apk" target="_blank">Скачать с GitHub</a> <QRCodeButton link="https://github.com/MatsuriDayo/NekoBoxForAndroid/releases/download/1.4.2/NekoBox-1.4.2-armeabi-v7a.apk" />

**Важно:** не скачивайте NekoBox из Google Play. Разработчик предупреждает, что версия оттуда — неофициальная и может быть опасной.
