Откройте установленный клиент и импортируйте конфигурацию.

### Windows (v2rayN)

1. Скачайте файл с предустановками (для РФ или для других стран)
2. В клиенте откройте `Settings` → `Restore and Backup` → `Restore from local` и выберите скачанный файл. Программа перезапустится
3. Скопируйте свою ссылку-подписку и нажмите `Ctrl + V` в окне клиента
4. Обновите подписку: `Subscription Group` → `Update subscriptions without proxy`

<div class="tip">

**Совет:** если `Ctrl + V` не сработало, добавьте подписку вручную: `Subscription Group` → `Subscription group settings` → `Add`. В поле `Remarks` введите название, в поле `URL` — ссылку-подписку.

</div>

### Android (NekoBox)

1. Скачайте файл с предустановками
2. В приложении откройте `Инструменты` → `Бэкап` → `Импорт из файла` и выберите скачанный конфиг
3. Перейдите в `Группы`, добавьте новую группу-подписку и вставьте свою ссылку-подписку (отдельную от ПК)
4. Перейдите в `Маршруты`, нажмите на три точки сверху, выберите `Управление ресурсами маршрутов` и нажмите «Обновить» у обоих файлов

### iOS (V2Box)

1. В приложении перейдите на вкладку `Configs`
2. Нажмите `+` вверху и выберите `Add Subscription`
3. Введите любое название в поле `Name`, вставьте ссылку-подписку в поле `URL` и нажмите `Add Subscribe`
4. Нажмите на появившуюся подписку — она будет использоваться для подключения

<div class="warning">

**Важно:** используйте разные подписки для каждого устройства. 1 подписка = 1 устройство.

</div>
