title: Насколько это безопасно?
order: 3
---

Соединение защищено на нескольких уровнях:

1. **TLS 1.3** — все данные шифруются с использованием [самого современного стандарта](https://en.wikipedia.org/wiki/Transport_Layer_Security#TLS_1.3) шифрования
2. **Отсутствие логов** — сервер _не хранит_ историю подключений и посещённых сайтов
3. **Маскировка трафика** — для внешнего наблюдателя трафик неотличим от обычного HTTPS
4. **Уникальный UUID** — каждый пользователь получает собственный идентификатор для аутентификации

### Что видит ваш провайдер?

| Без VPN                 | С VLESS VPN                        |
| ----------------------- | ---------------------------------- |
| Все посещённые сайты    | Подключение к обычному веб-серверу |
| DNS-запросы             | Зашифрованный TLS-трафик           |
| Объём трафика по сайтам | Общий объём трафика                |

<div class="warning">

**Важно:** не передавайте ваши данные для подключения (`UUID`, ссылку-подписку) третьим лицам — это может скомпрометировать безопасность вашего соединения.

</div>

> Шифрование TLS 1.3 обеспечивает _perfect forward secrecy_ — даже если ключ сервера будет скомпрометирован в будущем, ранее перехваченный трафик расшифровать не удастся.
